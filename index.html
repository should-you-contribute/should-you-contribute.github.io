Should I Contribute?
Don't have a project to test? Find one!

Checklist

Activity
How many commits have been made in the last month?
How many contributors are there to this project?
Responsiveness
Do reported issues get replies?
What percentage of pull requests are merged?
Documentation & Organization
Does the repository have a README?
Does the issue tracker label issues as good for newcomers?

Reach out!

This is a simple tool with quite a few limitations! No matter what the checklist looks like, if you're interested in contributing to a project, you should reach out to the community.

If you're not familiar with open source communications tools, try these other resources: video on open source communication, communication tools slides, communications norms slides. Or chat with the open source community on Twitter by pinging @openhatch, @outreachy, @codenewbies, or @opensourceway.
